# GCN Matrix - Type A - Imp00003


### About this document / Introduction

The purpose of this document is to provide some information about
GCN Matrix - Type A - Imp00003.


### About GCN Matrix - Type A - Imp00003

The purpose of this implementation is to predominantly use version 1.11 of
 JagStudio and the dialect of the BASIC programming language which is used
 by version 1.11 of JagStudio to develop versions of GCN Matrix - Type A
 that will run on hardware and software that is compatible with Atari Jaguar systems.
